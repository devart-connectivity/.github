## Build ORM models visually with Entity Developer

dotConnect data providers support EF Core, Entity Framework, NHibernate, and LINQ, delivering essential ORM capabilities for .NET application development. By integrating dotConnect with [Entity Developer](https://www.devart.com/entitydeveloper/), you can take advantage of visual ORM modeling to streamline workflows and minimize manual coding.
